API Used: Open Weather Map API

Open Weather Map takes in a location as a string and outputs the current temperature
and weather of that location.

Found on: https://openweathermap.org/api

Options Used

The options I used for this API were the temperature, and the exact type of weather
currently. The weather returns and ID which corresponds to a specific weather pattern
such as rain, snow, or scattered clouds.

Use Cases

Weather is useful for a veriety of different sites. It can be used for a hotel site to
persuade people to book during sunny days or for other applications
such as on a hiking website so you know when the best time to hike is.
